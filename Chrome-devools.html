4.2 Chrome devools

Chrome and other browsers provide a panel for troubleshooting your code.

  1. Instsall Chrome (if you don’t have it already).
  2. Read through [the documentation for Chrome Devtools][chrome-devools]

Explanation: This panel helps you find and fix issues with your front-end code quickly. The tool gets even more necessary when debugging complicated front-end setups.
