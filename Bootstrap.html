3.4 Bootstrap

Bootstrap is a bunch of CSS that people from around the world have already written. By using Bootstrap, you’ll save time writing CSS.

  1. Install Bootstrap (using <link>, not npm!).
  2. Try out some styles that sound interesting to you.

Bootstrap is one example of a CSS library. There are other CSS libraries but lots of sites use Bootstrap.
